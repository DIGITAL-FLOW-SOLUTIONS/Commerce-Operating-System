🧠 2. INPUT DETECTION ENGINE
🎯 Purpose:

Identify what the user entered

📦 Model: BuildSession
Fields:
id
user_id (nullable)
input_value
input_type (facebook, instagram, tiktok, manual)
status (processing, completed, failed)
extracted_data (JSON)
created_at
⚙️ Detection Logic
Condition	Type
contains “facebook.com”	Facebook
contains “instagram.com” or “@”	Instagram
contains “tiktok.com” or “@”	TikTok
else	Business Name

🟢 4. PRODUCT INFERENCE ENGINE (VERY IMPORTANT)
🎯 Purpose:

Turn posts into products

⚙️ Logic

From caption:

Detect product name
Detect price (KES patterns like “KSh 1500”)
Use image as product image
Example:

Caption:

“New heels KSh 2500 available”

→ becomes:

name: “New heels”
price: 2500
🤖 AI PROMPT
Design a product inference system:

- Input: post caption + image
- Extract:
  - Product name
  - Price (KES detection)
  - Description
- Handle missing price gracefully
- Generate clean product objects
